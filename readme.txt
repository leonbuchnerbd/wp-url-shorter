== Beschreibung ==
URL-Shorter ist ein Plugin zur Verkürzung von URLs mit integriertem Klicktracking und QR-Code Generierung. Über das Admin-Panel lassen sich verkürzte URLs erstellen, bearbeiten und QR-Codes herunterladen.

== Installation ==
1. Lade den Plugin-Ordner in das Verzeichnis /wp-content/plugins/ hoch oder lade die ZIP File hoch.
2. Aktiviere das Plugin im WordPress-Dashboard.
3. Nutze das Admin-Menü "URL-Shorter" zur Verwaltung der URLs.
